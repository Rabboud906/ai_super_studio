# 🚀 AI Super Studio

A Flask-based AI web application that integrates multiple Hugging Face models to provide:

- 💬 Chat with an AI assistant  
- 🎨 Image generation from text prompts  
- 📄 PDF question-answering using extracted content  

---

## 📌 Overview

AI Super Studio is a full-stack web application designed to demonstrate the power of modern AI APIs. It combines natural language processing, computer vision, and document understanding into a single interactive platform.

Users can:
- Chat with an AI model in real time
- Generate images from text prompts
- Upload PDFs and ask questions about their content

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **AI Models:** Hugging Face Inference API
- **Frontend:** HTML, CSS (custom styling)
- **Libraries:**
  - `huggingface_hub`
  - `pypdf`
  - `python-dotenv`
  - `Pillow`
  - `requests`
